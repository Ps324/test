# Nujeens Flucht – Webgame

Interaktives Browser-Spiel mit Entscheidungen, Zufall und Ressourcen.

## Start

Lokal einfach `webgame/index.html` im Browser öffnen.

## Dateien

- `index.html`
- `style.css`
- `script.js`
- `.github/workflows/deploy.yml`

## GitHub Pages

Die Action veröffentlicht bei jedem Push auf `main` den Ordner `webgame` auf dem Branch `gh-pages`.
